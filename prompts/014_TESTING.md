# Testing Prompt

Use this prompt for test planning, test implementation, and quality validation work.

## Guidance

- Favor real behavior and meaningful assertions.
- Cover core flows, edge cases, and regression risks.
- Keep tests maintainable and focused on outcomes.

## Output Expectations

- Describe the feature or bug being tested.
- Note the test strategy, cases, and expected results.
- Mention any setup, tooling, or follow-up validation.
